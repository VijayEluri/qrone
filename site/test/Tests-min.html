CSS Sprite : <img>

CSS Sprite is a well known technique for optimizing which reduces HTTP Request while loading web pages. This makes faster the web page.

You have to do isags to make it use background-image and background-position with measuring image size and lo creating image which concatenates all images in your web site and rewrite all <img> tcation of your concatenated image, instead of simple src attributes.

syntax

<img src="file">

sample

All you need to write is image tag. QrONE do that automatically.

<img src="image.png" align="absmiddle">

QrONE will measure your images and add images to concatenated image, then rewrite your tag using background properties.

<img src="dummy.png" style="width:67px;height:39px;background: no-repeat 0px -881px url(isprite.png);" align="absmiddle">

CSS : scale9 property

'scale9' property is a very powerful custom property proposed by QrONE minifier.
You need just 1 image to create well-desinged box like rounded-coner drop-shadowed box with floating icon to top-right corner very easily.

This applies scalable background image to a box like Adobe Flex's scale-9 formatting.

syntax

Use in style attribute only, specified in style tags or stylesheet will be ignored.

scale9:url(file) leftX rightX topY bottomY;

scale9:url(box.png) 10px 20px 30px 40px;
// scale-9 formatting in HTML.

scale9:url(box.png) 10px 20px;
// scale-9 with horizontal scaling only.

sample

You need only 1 image and 1 css rule to create this code box.

<div style="scale9:url(box.png) 15px 50px 15px 25px;">
contents inside.
</div>
