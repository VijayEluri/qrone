CSS : scale9 property

'scale9' property is a very powerful custom property proposed by QrONE minifier.
You need just 1 image to create well-desinged box like rounded-coner drop-shadowed box with floating icon to top-right corner very easily.

This applies scalable background image to a box like Adobe Flex's scale-9 formatting.

syntax

Use in style attribute only, specified in style tags or stylesheet will be ignored.

scale9:url(file) leftX rightX topY bottomY;

scale9:url(box.png) 10px 20px 30px 40px;
// scale-9 formatting in HTML.

scale9:url(box.png) 10px 20px;
// scale-9 with horizontal scaling only.

sample

You need only 1 image and 1 css rule to create this code box.

<div style="scale9:url(box.png) 15px 50px 15px 25px;">
contents inside.
</div>
box.png
